# User Guide Screenshot Checklist

Place sanitized user-guide screenshots in this directory.

Expected files:

1. `01-release-download.png` - OutlookQuickMove release entry on the GitHub Releases page.
2. `02-quick-move-tab.png` - Outlook ribbon showing the `Quick Move` tab.
3. `03-quick-move-dialog.png` - `Quick Move` dialog with filtered folder results.
4. `04-undo-quick-move.png` - `Undo Quick Move...` dialog.
5. `05-settings-data-files.png` - `Settings` dialog, `Data Files` tab.
6. `06-settings-frequent-folders.png` - `Settings` dialog, `Frequent Folders` tab.
7. `07-settings-undo-history.png` - `Settings` dialog, `Undo History` tab.
8. `08-add-to-quick-access-toolbar.png` - Quick Access Toolbar menu showing `More Commands...`.
9. `09-quick-access-toolbar-shortcut.png` - Customize Quick Access Toolbar settings with `Quick Move` added.

Privacy requirements before committing screenshots:

- No real names, email addresses, avatars, or work accounts.
- No mail subjects, sender names, recipient names, or message previews.
- No private mailbox, PST, OST, shared mailbox, customer, project, or ticket names.
- No Windows usernames, local paths, or network paths.
- Use a test Outlook profile where possible. Otherwise crop or blur sensitive areas.
